# DriveZone - Premium Car Showroom Management System

DriveZone is a robust, premium Java Web Application built for managing car dealership inventories and handling customer enquiries. It offers a sleek, modern, responsive user interface with dynamic Light/Dark mode functionality.

## 🚀 Key Technologies
*   **Frontend**: HTML5, CSS3 (Custom Variables), JavaScript, FontAwesome
*   **Backend**: Java Servlets, JSP (JavaServer Pages)
*   **Database**: MySQL (via XAMPP)
*   **Server**: Apache Tomcat

## ✨ Features Overview
*   **Role-Based Access Control**: Secure login system differentiating between regular users and Administrators.
*   **Inventory Management**: Admins can add, edit, and remove vehicles dynamically.
*   **Dynamic Theming**: Seamless Light and Dark mode toggle preserving user preference.
*   **Customer Enquiries**: Users can submit purchase or general enquiries for specific vehicles.
